# Trak : TrashKonek

## Requirements:

1. Nodejs v16
2. MongoDB v4.4.2 or higher


## Installation

1. npm install
2. create an .env file using the env.example
3. run 'nodemon' in the console

## API

1. /api/user/register - { "email" : "", "password": "" }
2. /api/user/auth - { "email" : "", "password": "" }
3. /api/user - { "id" : "" }
